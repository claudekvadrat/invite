# Приглашение на юбилей · Сапуре 75 лет

Одностраничный пригласительный сайт, оптимизированный под смартфоны.
Статический — никакой сборки не требуется. Vercel раздаёт файлы как есть.

## Структура

| Файл          | Что внутри                                            |
|---------------|-------------------------------------------------------|
| `index.html`  | разметка приглашения                                  |
| `styles.css`  | всё оформление (красно-золотой стиль)                 |
| `script.js`   | обратный отсчёт, анимации, лепестки роз               |
| `img/main.jpg`| фотография юбиляра                                    |
| `vercel.json` | настройки раздачи и кеширования для Vercel            |

## Как поменять данные

Всё редактируется в `index.html`:

- **Имя** — блок `<h1 class="name">Сапура</h1>`
- **Дата** — блок «Дата» → `19 сентября 2026`
- **Время** — блок «Начало» → `12:00`
- **Место** — блок «Место торжества»
- **Ссылка 2ГИС** — атрибут `href` у кнопки «Открыть в 2ГИС»

⚠️ Дата обратного отсчёта задаётся отдельно в `script.js`:
строка `var target = new Date("2026-09-19T12:00:00+05:00")`.
Если поменяете дату праздника — поправьте её и здесь (часовой пояс +05:00 — Казахстан).

Чтобы заменить фото — положите новый файл в `img/main.jpg`
(вертикальное, портретной ориентации смотрится лучше всего).

## Деплой на Vercel

Ничего настраивать не нужно. При каждом `git push` в подключённый
репозиторий Vercel автоматически публикует новую версию.
Framework Preset в настройках проекта Vercel — **Other** (статический сайт).
